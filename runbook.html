1. Email Manager active

File ~/email_manager.py (1,745 lines)
Port 5000
Stack Python 3 · Flask · IMAP · Mistral-7B

Scans iCloud inbox (46k+ emails), categorizes into 9 buckets (security, job, finance, shopping, travel, social, news, subscription, other), surfaces unsubscribe candidates and spam via AI analysis, and generates weekly digests. Full-text search, batch delete, CSV export.

04 llama.cpp Setup

Dir ~/llama.cpp/
Binary ~/llama.cpp/build/bin/llama-server (CUDA-enabled)

Compiled GGML inference server. Serves OpenAI-compatible API endpoints.

Models in ~/models/:

FileSizeUse
mistral-7b-instruct-v0.2.Q4_K_M.gguf4.3 GBChat/completions (port 8080)
nomic-embed-text-v1.5.Q4_K_M.gguf84 MBText embeddings for RAG (port 8081)
nomic-embed — embeddings, port 8081 
cd ~/llama.cpp
./build/bin/llama-server \
  -m ~/models/nomic-embed-text-v1.5.Q4_K_M.gguf \
  --port 8081 --host 127.0.0.1 --embedding -ngl 999
mistral-7b — chat, port 8080 
cd ~/llama.cpp
./build/bin/llama-server \
  -m ~/models/mistral-7b-instruct-v0.2.Q4_K_M.gguf \
  -ngl 999 -c 4096 -b 256 -t 10 \
  -ctk q8_0 -ctv q8_0 -fa 1 \
  --host 127.0.0.1 --port 8080
load any other gguf model 
cd ~/llama.cpp
./build/bin/llama-server \
  -m ~/models/<model-name>.gguf \
  -ngl 999 -c 4096 \
  --host 127.0.0.1 --port 8082

10 Known Issues

  • BUG-1
    Email manager scan drops at 600–1000 emails — IMAP connection timeout. Needs exponential backoff + per-email retry.
  • BUG-2
    Player card cache is stale after stat updates — cache key is tank_id only. Clear with rm -rf ~/tankpit-bot-legacy/cache/*.
  • BUG-3
    No cron for bible refresh — must run rebuild_bible.py manually after scraping new profiles.
  • BUG-4
    Leaderboard data not in RAG — leaderboards.json not merged into bible_chunks.json, so /ask can't answer tournament history questions accurately.
  • SEC-1
    Credentials in .env files — Discord token and TankPit password are in version-tracked .env files. Rotate if repo is ever made public.
